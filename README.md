# Strategic Advisor AI - Bannerlord

An advanced AI module for Mount & Blade II: Bannerlord that analyzes battles and automatically commands troops.

## Features

* 🧠 Dynamic AI decision-making
* ⚔ Automatic troop control (formations, flanks, charges)
* 🏰 Full siege support (attacker / defender)
* 🔄 Real-time strategy adaptation
* 📊 Learning system (brain.json)
* ⌨ Toggle AI with key **O**
* 🚫 Disabled in camps and naval battles

## Installation

1. Compile the project in **Release**
2. Copy `StrategicAdvisorAI.dll` into:

```
Mount & Blade II Bannerlord/Modules/StrategicAdvisorAI/bin/Win64_Shipping_Client/
```

3. Enable the mod in the launcher

## Controls

* **O** → Toggle AI auto-command ON/OFF

## How it works

The AI analyzes:

* Troop composition
* Terrain
* Enemy behavior (charges, formations)
* Numerical advantage

Then selects and executes a strategy dynamically.
